## Личный кабинет

Пользователь с активным доступом видит все страницы сайта. Пользователь с истёкшим доступом видит только страницу "Личный кабинет" и после логина попадает в раздел "Текущий доступ" личного кабинета.

### Профиль

#### Пользователь
* Имя пользователя
* Email
* Дата регистрации

#### Статистика использования
* Дата последнего входа
* Количество поисков за всё время
* Количество просмотренных картин

### Доступ
* Статус: Активен/Истёк
* Осталось дней доступа
* Дата окончания доступа

#### Продление доступа
* Поле ввода целого числа "Количество дней". Значения от 1 до 365.
* Стоимость выбранного кличества дней
* Кнопка "Оплатить"

#### История платежей
Таблица с полями: "Дата оплаты", "Количество оплаченных дней", "Сумма оплаты", "Способ оплаты"

### Настройки
* Email-уведомления об окончании доступа
* Язык интерфейса (Ru или En)

#### Смена пароля
* Текстовое поле ввода "Текущий пароль"
* Текстовое поле ввода "Новый пароль"
* Текстовое поле ввода "Повтор нового пароля"
* Кнопка "Сохранить"
