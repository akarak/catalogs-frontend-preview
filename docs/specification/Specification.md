## Личный кабинет

Пользователь с активным доступом видит все страницы сайта. Пользователь с истёкшим доступом видит только страницу "Личный кабинет" и после логина попадает в раздел "Текущий доступ" личного кабинета.

### Профиль
#### Пользователь
* Имя пользователя
* Email
* Дата регистрации
#### Статистика использования
* Дата последнего входа
* Количество поисков за всё время
* Количество просмотренных картин
### Текущий доступ
* Статус: Активен/Истёк
* Осталось дней доступа
* Дата окончания доступа
* Кнопка "Продлить доступ"
### История пополнений
Таблица с полями:
* Дата оплаты
* Количество купленных дней
* Сумма оплаты
* Способ оплаты
### Настройки
* Сменить пароль
* Email-уведомления об окончании доступа
* Язык интерфейса (Ru или En)
